## 그래프

그래프(Graph)는 **객체** 간의 **연결 관계**를 표현하는 자료구조이다.
객체는 정점(Vertex, Node)으로 표현하고 연결 관계는 간선(Edge)으로 표현한다.

ex) 인스타그램 팔로우 관계, 지하철 노선도

## 그래프 종류

1. 방향 그래프
   두 정점을 연결하는 간선에는 시작점과 끝점이 명확히 정의되어 있어 일방향 연결을 나타낸다.
2. 무방향 그래프
   무방향 그래프에서 간선은 양방향성을 가진다. 두 정점 어느 쪽에서든 다른 쪽으로 이동할 수 있다.
3. 다중 그래프
   두 정점 사이에 여러 개의 간선이 존재할 수 있고, self-loop도 허용된다.
4. 단순 그래프
   정점 간에 하나의 간선만 존재하며, self-loop가 허용되지 않습니다.
5. 가중치 그래프 (다익스트라, 플로이드-워셜)
   각 간선에 특정 값(가중치)이 할당됩니다. 이 가중치는 간선을 통한 이동 비용, 거리, 시간, 용량 또는 다른 어떤 수량적인 측정치를 나타낼 수 있다.
6. 비가중치 그래프
   간선에 특별한 가중치 정보가 부여되지 않고, 각 간선의 가중치를 굳이 구분하지 않습니다.
7. 순환 그래프
   그래프 내에 사이클이 존재하는 그래프입니다. 사이클은 시작 정점과 끝 정점이 같은 경로를 의미합니다.
8. 비순환 그래프 (Tree, DAG, 위상정렬)
   비순환 그래프는 그래프 내에 사이클이 존재하지 않는 그래프입니다. 방향 비순환 그래프 (DAG, Directed Acyclic Graph)는 특히 중요하다.
